# 2023S-AIKU-D2D
![](logo.png)
2023S-AIKU-D2D is the material (lecture notes, exercises and assignments) repository for 2023 Summer DeepintoDeep (D2D).\
D2D is a deep learning bootcamp for juniors learning and experiencing various deep learning and its applications with PyTorch at AIKU.

## Contents


More information is available at [curriculum.md](https://github.com/ONground-Korea/2023-AIKU_DeepLearning-Bootcamp/blob/main/curriculum.md)
